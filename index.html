Servidores de Internet

Conoce los diferentes tipos de servidores y sus funciones

DNS

Servidor DNS

Texto breve: Sistema que traduce nombres de dominio a direcciones IP, como una guía telefónica de Internet.

¿Qué es y para qué sirve?

El DNS (Domain Name System) es un sistema que traduce nombres de dominio legibles por humanos (como www.google.com) a direcciones IP numéricas que las computadoras pueden entender. Funciona como una "guía telefónica" de Internet, permitiendo que navegues usando nombres fáciles de recordar en lugar de números complicados.

Ejemplo: Cuando escribes www.google.com en tu navegador, el servidor DNS lo traduce automáticamente a la dirección IP 142.250.191.14, que es la ubicación real del servidor de Google.
WEB

Servidor WEB

Texto breve: Almacena y entrega páginas web, imágenes y contenido a los usuarios que navegan por Internet.

¿Qué es y qué función cumple?

Un servidor web almacena, procesa y entrega páginas web, imágenes, videos y otros recursos web a los usuarios. Cuando solicitas una página web, el servidor procesa tu solicitud HTTP/HTTPS y te envía el contenido correspondiente.

Funciones principales:

  • Almacenar archivos web (HTML, CSS, JavaScript, imágenes)
  • Procesar solicitudes de los navegadores
  • Ejecutar aplicaciones web dinámicas
  • Gestionar bases de datos
Ejemplo: Los servidores que alojan las páginas que visitas diariamente como Facebook, YouTube, Netflix, Wikipedia, Instagram, y cualquier sitio web que uses.
FTP

Servidor FTP

Texto breve: Permite transferir archivos entre computadoras, ideal para subir y descargar archivos por Internet.

¿Para qué se usa?

FTP (File Transfer Protocol) permite transferir archivos entre computadoras a través de Internet de forma segura y eficiente. Se utiliza tanto para subir archivos desde tu computadora a un servidor como para descargar archivos desde un servidor a tu computadora.

Usos comunes:

  • Desarrolladores web suben archivos a sus sitios web
  • Descargar software desde repositorios oficiales
  • Backup y sincronización de archivos
  • Compartir archivos grandes entre empresas
Ejemplo: Un diseñador web que sube las imágenes y archivos HTML de un sitio web al servidor de hosting, o descargar programas desde servidores FTP de universidades o empresas de software.
DHCP

Servidor DHCP

Texto breve: Asigna automáticamente direcciones IP a dispositivos en la red, evitando configuraciones manuales.

¿Qué hace con las direcciones IP?

DHCP (Dynamic Host Configuration Protocol) asigna automáticamente direcciones IP y configuración de red a dispositivos que se conectan a una red. Esto evita conflictos de IP y facilita enormemente la administración de redes grandes y pequeñas.

Configuraciones que asigna:

  • Dirección IP única para cada dispositivo
  • Máscara de subred
  • Gateway predeterminado (puerta de enlace)
  • Servidores DNS
Ejemplo: El router en tu casa funciona como servidor DHCP y asigna automáticamente direcciones IP como 192.168.1.100 a tu celular, 192.168.1.101 a tu laptop, 192.168.1.102 a tu Smart TV, etc., sin que tengas que configurar nada manualmente.
MAIL

Servidor de Correos

Texto breve: Gestiona el envío y recepción de correos electrónicos usando protocolos SMTP, IMAP y POP3.

Diferencia entre enviar y recibir correos:

Los servidores de correo manejan el envío, recepción y almacenamiento de emails. Utilizan diferentes protocolos según la función que realicen:

SMTP (Simple Mail Transfer Protocol)

Función: ENVIAR correos

Se encarga de transportar los emails desde tu cliente de correo hacia el servidor del destinatario.

IMAP (Internet Message Access Protocol)

Función: RECIBIR y sincronizar correos

Permite acceder a los correos desde múltiples dispositivos manteniendo sincronización entre todos.

POP3 (Post Office Protocol)

Función: DESCARGAR correos

Descarga los correos a un solo dispositivo, generalmente eliminándolos del servidor después de la descarga.

Ejemplo: Gmail utiliza servidores SMTP para enviar tus correos y servidores IMAP para que puedas leer tus emails tanto en tu celular como en tu computadora, manteniendo todo sincronizado.
